# Apollo_or_Autoware
一份Apollo和Autoware(ROS)的对比


# 拉取、修改、提交、合并流程

## 拉取

```bash
git clone https://github.com/qpc001/Apollo_or_Autoware.git
```

## 如果你需要修改文档，则需要创建你的分支

```bash
git branch 你的分支名称
git checkout 你的分支名称
```


## 进行你自己的修改

...

## 提交你的分支

```bash
git add .
```

```bash
git commit -m "你修改部分的简要备注"
```

```bash
git remote add origin https://github.com/qpc001/Apollo_or_Autoware.git
```

开始提交

```bash
git push origin 你的分支名称
```
